# Reines Wasser – Strategie- und Lead-Page

Dieses Projekt liefert eine moderne Landingpage für den strategischen
Marktbericht 2026 zur **dezentralen Wasserfiltration** (PFAS-Remediation,
mieterfreundliche Filtersysteme, Infrastruktur-Optimierung). Ziel ist eine
vertrauensbildende Plattform mit klarer Nutzerführung von Daten-Insights bis
zur Lösung.

## Inhalte der Seite

- **Strategischer Überblick**: Markt- und Regulierungsdruck durch die
  Trinkwasserverordnung 2026 (PFAS-20).
- **Prioritäten-Nische**: High-End Filtration, Fokus auf Mieter und lokale
  Wasserqualität.
- **Monetarisierung**: High-Ticket Systeme + wiederkehrende Kartuschen-Umsätze.
- **Blueprint**: PFAS-Radar, Validierungs-Hub, Mieter-Guide, Myth-Busting.
- **Risiko-Management**: Rechtssichere Kommunikation ohne Angstmarketing.

## Dateien

- **index.html**: Vollständige Landingpage-Struktur in Deutsch.
- **styles.css**: Modernes UI (Gradients, Cards, responsive Grids, CTAs).
- **script.js**: Smooth-Scroll für Navigation und CTA-Buttons.
